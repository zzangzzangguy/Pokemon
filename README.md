# Pokemon

> 포켓몬 TCG 카드 검색 및 관리 애플리케이션

 ### 사용자가 포켓몬 TCG 카드를 쉽게 검색하고 관리할 수 있도록 도와주는 iOS 애플리케이션입니다. 이 애플리케이션을 통해 사용자는 다양한 포켓몬 카드를 탐색하고, 자신이 좋아하는 카드를 즐겨찾기에 추가할 수 있습니다.
